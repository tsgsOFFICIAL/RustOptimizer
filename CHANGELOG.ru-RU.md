# Список изменений

Здесь документируются все значимые изменения в Rust Optimizer.

## Unreleased

## 0.8.6
- Удалена отдельная страница «Конфигурации» - она объединена с новой страницей резервного копирования, поскольку обе страницы, по сути, занимались одним и тем же: управлением cfg-файлами Rust.
- Добавлена страница резервного копирования: переключение между «Настройками» (`client.cfg`) и «Клавишами» (`keys.cfg`), просмотр истории резервных копий для каждого типа, создание резервной копии с собственным названием (или оставьте поле пустым для метки времени), а также восстановление или удаление любой записи - оба действия сначала запрашивают подтверждение, так как удаление нельзя отменить.
- Резервные копии теперь создаются автоматически перед применением готового профиля и перед восстановлением, которое перезаписывает текущий файл - это заменяет старую единственную копию `client.cfg.bak`, которая хранила только одну версию.
- Кнопки предустановленных профилей на панели управления теперь заблокированы, если Rust не установлен, вместо того чтобы оставаться кликабельными без какого-либо эффекта.
- Карточка «Накопители» на странице «Система» теперь обновляется каждые несколько секунд, а не только при открытии страницы.
- Исправлено: фон кнопок при наведении по всему приложению показывал резкие прямые углы вместо повторения скруглённой формы самой кнопки.

## 0.8.5
- Добавлена страница «Система» (доступна из боковой панели): текущая загрузка и характеристики процессора/видеокарты/ОЗУ, материнская плата/BIOS, накопители и данные ОС, а также настройки уровня ОС - схема электропитания, точность указателя, Игровой режим, фоновая запись и полноэкранные оптимизации для Rust.
- Добавлены значки предупреждения рядом с любой настройкой на странице «Система», которая не соответствует рекомендуемому значению, с подсказкой, объясняющей почему - включая нехватку ОЗУ (менее 16 ГБ), работу ОЗУ ниже номинальной скорости (XMP/EXPO не включён в BIOS) и нехватку свободного места на диске Rust (менее 10%).
- Карточка «Система» в обзоре оптимизации на панели управления теперь связана с реальными настройками страницы «Система» вместо жёстко заданного значения «12 / 16 settings», и получила трёхуровневый статус (красный/жёлтый/зелёный) вместо простого деления хорошо/не оптимизировано.
- Затемнены карточки «Производительность», «Сеть» и «Графика» в обзоре оптимизации (пока тестовые данные) - теперь они выглядят так же, как другие ещё не реализованные функции, и только карточка «Система» выглядит интерактивной.
- Карточка «Система» в обзоре оптимизации теперь кликабельна и сразу открывает страницу «Система» - то же место, куда ведёт строка «Подробнее» в карточке «Информация о системе».
- Исправлено: подсказка у значков предупреждения иногда появлялась только после того, как курсор несколько раз перемещали туда-сюда.
- Исправлено: счёт «Система» на панели управления не обновлялся после изменения настройки на странице «Система» до перезапуска приложения.
- Добавлена поддержка ~~зачёркнутого текста~~ в отображении списка изменений.

## 0.8.4
- Добавлено настоящее определение установки Rust: приложение теперь находит реальную папку установки Rust через Steam, поэтому кнопки "Запустить Rust" и "Проверить файлы игры" корректно отключаются (с красным индикатором состояния на боковой панели), если Rust не установлен, вместо того чтобы всегда считать его установленным.
- Добавлены рабочие предустановленные профили: Слабый ПК, Соревновательный, Стример и Кинематографичный теперь действительно изменяют настройки графики/производительности Rust в `client.cfg` (сначала создавая резервную копию оригинала) вместо того, чтобы быть нефункциональными заглушками.
- Удалён раздел "Быстрая оптимизация", который дублировал предустановленные профили, не добавляя ничего нового.

## 0.8.3
- Исправлено: панель управления выбирала встроенную видеокарту вместо дискретной (и показывала её загрузку) на системах с обеими, например AMD APU вместе с картой Radeon.
- Определение имени процессора/видеокарты и объёма ОЗУ переведено с WMI и Win32 API памяти на тот же бэкенд LibreHardwareMonitor, который уже используется для текущей загрузки.

## 0.8.2
- Добавлена реальная информация о системе в карточку "Информация о системе" на панели управления: настоящая модель и текущая загрузка процессора/видеокарты, а также используемая/общая память, вместо старых условных значений.
- Добавлены плавные переходы при наведении и нажатии на кнопки, а также исправлена цветовая вспышка, которая кратковременно появлялась при переключении активного пункта боковой панели, особенно заметная в тёмной теме.
- Исправлено: текст в списке изменений выходил за край окна вместо переноса строк.
- Добавлена поддержка цитат и блоков кода в отображении списка изменений, а встроенный `код` теперь получает настоящий фон и рамку вместо простой смены шрифта.

> Вот как теперь выглядит цитата.

```
блоки кода теперь отображаются
в рамке моноширинным шрифтом
```

## 0.8.1
- Добавлена настоящая кнопка "Запустить Rust", которая запускает игру через Steam и автоматически отключается, пока Rust уже запущен.
- Добавлены недостающие переводы интерфейса панели управления, боковой панели и настроек на датский и русский языки, которые ранее отображались на английском.

## 0.8.0

- Добавлен интерфейс панели управления с боковой навигацией: обзор системы, профили быстрой оптимизации и предустановленные профили (пока используются тестовые данные до появления реального определения системы).
- Переработана страница настроек: выбор темы (Светлая/Тёмная/Системная) и языка теперь представлен в виде значков.
- Улучшена страница «О программе»: добавлена кнопка ручной проверки обновлений, а также ссылки на GitHub, Discord и Ko-fi.
- Набор иконок заменён на Phosphor Icons (с Simple Icons для логотипов брендов), что значительно уменьшило размер загружаемого приложения.

## 0.7.0

- Добавлен установщик для Windows (`Setup.exe`) как альтернатива портативным zip-архивам, устанавливается для текущего пользователя с ярлыками в меню «Пуск» и опциональным ярлыком на рабочем столе.
- Добавлена автоматическая проверка обновлений: при запуске проверяется наличие новой версии на GitHub, и если она найдена, показываются номер версии и список изменений прямо в окне, прежде чем принять решение об обновлении.
- Добавлено обновление в один клик: новая версия загружается и применяется на месте (для установленных копий повторно запускается установщик, для портативных копий заменяются файлы), после чего приложение автоматически перезапускается.
