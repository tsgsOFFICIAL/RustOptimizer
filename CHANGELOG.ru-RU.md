# Список изменений

Здесь документируются все значимые изменения в Rust Optimizer.

## Unreleased

## 0.8.1
- Добавлена настоящая кнопка "Запустить Rust", которая запускает игру через Steam и автоматически отключается, пока Rust уже запущен.
- Добавлены недостающие переводы интерфейса панели управления, боковой панели и настроек на датский и русский языки, которые ранее отображались на английском.

## 0.8.0

- Добавлен интерфейс панели управления с боковой навигацией: обзор системы, профили быстрой оптимизации и предустановленные профили (пока используются тестовые данные до появления реального определения системы).
- Переработана страница настроек: выбор темы (Светлая/Тёмная/Системная) и языка теперь представлен в виде значков.
- Улучшена страница «О программе»: добавлена кнопка ручной проверки обновлений, а также ссылки на GitHub, Discord и Ko-fi.
- Набор иконок заменён на Phosphor Icons (с Simple Icons для логотипов брендов), что значительно уменьшило размер загружаемого приложения.

![тестовые данные](https://raw.githubusercontent.com/tsgsOFFICIAL/RustOptimizer/master/Assets/Changelog/0.8.0%20mock%20ui.png)

## 0.7.0

- Добавлен установщик для Windows (`Setup.exe`) как альтернатива портативным zip-архивам, устанавливается для текущего пользователя с ярлыками в меню «Пуск» и опциональным ярлыком на рабочем столе.
- Добавлена автоматическая проверка обновлений: при запуске проверяется наличие новой версии на GitHub, и если она найдена, показываются номер версии и список изменений прямо в окне, прежде чем принять решение об обновлении.
- Добавлено обновление в один клик: новая версия загружается и применяется на месте (для установленных копий повторно запускается установщик, для портативных копий заменяются файлы), после чего приложение автоматически перезапускается.

## 0.6.1

- Исправлена проблема, из-за которой изображения и GIF в списке изменений становились слишком большими при увеличении ширины окна: теперь их размер отображения ограничен, а не растягивается на всю доступную ширину.

## 0.6.0

- Добавлен встроенный просмотрщик списка изменений, чтобы обновления объясняли *почему* они произошли, а не просто сообщали о новом номере версии.
- Добавлена поддержка изображений и анимированных GIF в списке изменений, декодируемых покадрово без дополнительных зависимостей.

![Демонстрация переключения языка](https://raw.githubusercontent.com/tsgsOFFICIAL/RustOptimizer/master/Assets/Changelog/demo-0.6.0.gif)

## 0.5.0

- Исправлена ошибка, из-за которой сборки, зависящие от framework, включали файлы автономной среды выполнения.
- Изменена версия продукта, используемая для версии файла, отображаемой в заголовке окна.

## 0.4.0

- Добавлен собственный безрамочный элемент управления `TitleBar` для Windows, заменяющий системную строку заголовка.
- Добавлена поддержка локализации (английский, датский, русский) с автоматическим определением системного языка.
- Добавлено переключение между светлой/тёмной/системной темой.
- Добавлен полный набор иконок IconPacks.Avalonia через подмодуль (до выхода официальной версии для Avalonia 12).
